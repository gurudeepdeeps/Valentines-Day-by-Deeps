Love Login by Deeps is a static website project that provides a simple and effective login system for your website. This project uses HTML, CSS, and JavaScript to create a user-friendly interface for users to login and access restricted content. The project is designed to be easy to set up and customize, making it perfect for developers and non-developers alike.

The project consists of four HTML files: `book.html`, `main.html`, `index.html`, and a JavaScript file called `script.js`. The `book.html` file serves as a placeholder for the login page, while the `main.html` file is the page that users are redirected to after a successful login. The `index.html` file is the main page of the website, and the `script.js` file handles the login functionality.
